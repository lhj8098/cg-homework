## 一、实验目标

1. 理论区分：理解光线投射（Ray Casting）与光线追踪（Ray Tracing）二者本质区别；

2. 全局光照：通过发射次级射线实现硬阴影、理想镜面反射效果，掌握全局光照实现逻辑；

3. GPU 编程优化：把传统递归光线追踪改写为循环迭代模式，适配 GPU 并行计算特性。

## 二、实验原理

本实验使用 Whitted\-Style 光线追踪模型，主光线打到物体表面后分两步处理：

1. 阴影检测：从交点向光源发射暗影射线，射线中途碰到其他物体则该点处于阴影，仅渲染环境光；

2. 材质分支逻辑

- 漫反射材质：使用 Phong 模型计算像素颜色，光线传播直接终止；

- 镜面材质：通过反射公式计算反射光线方向，生成新反射射线继续弹射，直到碰到漫反射物体或达到最大弹射次数。

### 反射向量公式

$\mathbf{R} = \mathbf{L}_{in} - 2(\mathbf{L}_{in} \cdot \mathbf{N})\mathbf{N}$
$\mathbf{L}_{in}$：入射光线方向；$\mathbf{N}$：物体表面单位法向量；$\mathbf{R}$：反射光线方向。

### 迭代式光线追踪说明

GPU 不适合递归运算，采用循环模拟光线多次弹射：

1. 初始化光线衰减系数`throughput=1.0`、最终颜色`final_color=0`；

2. 光线击中镜面物体：更新光线起点、方向，衰减反射亮度，继续循环；

3. 光线击中漫反射物体：计算当前光照颜色，乘以衰减系数叠加到最终颜色，跳出循环；

4. 循环上限由最大弹射次数控制，避免无限迭代。

### Shadow Acne 精度问题说明

浮点数误差会导致次级射线与物体自身相交，画面出现黑色噪点。
解决方案：次级射线起点沿法线向外偏移极小数值：
$\mathbf{P}_{new} = \mathbf{P} + \mathbf{N} \times \epsilon \quad (\epsilon=1e-4)$

## 五、实验任务与代码实现逻辑

### 任务 1：隐式构建三维场景 \+ 材质区分

不加载外部模型，纯数学定义几何体，使用材质 ID 区分渲染类型：

1. 地面平面：y=\-1\.0，法向量 \(0,1,0\)，黑白棋盘格纹理，漫反射材质；

2. 红色漫反射球：坐标 \(\-1\.5,0,0\)，半径 1\.0，仅漫反射无反射；

3. 银色镜面球：坐标 \(1\.5,0,0\)，半径 1\.0，理想镜面材质，支持多级反射。

### 任务 2：迭代循环实现光线弹射

1. 循环次数限制最大弹射次数；

2. throughput 记录光线多次反射后的亮度衰减；

3. 击中镜面则更新光线参数继续循环；击中漫反射则计算光照并终止光线。

### 任务 3：硬阴影实现与精度 bug 修复

1. 渲染漫反射表面时生成暗影射线，检测遮挡关系实现硬阴影；

2. 所有反射射线、暗影射线起点沿法线偏移极小值，解决自相交黑斑噪点。

### 任务 4：交互 UI 面板

使用 ti\.ui\.Window 绘制窗口，绑定光源坐标、最大弹射次数滑动条，拖动滑块实时重渲染画面。

## 七、实验现象与结果分析

1. 弹射次数对比

- Max Bounces=1：无镜面反射，仅基础光照 \+ 硬阴影；

- Max Bounces≥2：镜面球体出现嵌套反射，数值越高反射层数越多。

2. 光源调节效果
拖动光源三轴滑块，场景阴影同步移动，阴影边缘锐利，为硬阴影效果。

3. 材质表现差异
红球只有基础漫反射，无反射成像；银球可完整反射地面、红球与环境。

4. 精度修复效果
开启法线偏移后无黑斑噪点，彻底解决光线自相交 bug。

5. 拓展效果
折射材质可观察光线穿透物体的透视偏移；MSAA 可平滑模型锯齿边缘。
