2. **数学基础**：熟练三维向量运算，掌握表面法向量、光线方向、视线向量、反射向量的求解与归一化操作。

## 二、实验原理

### 2\.1 Phong 光照总公式

物体表面最终像素颜色由三类光分量叠加得到：
$I = I_{ambient} + I_{diffuse} + I_{specular}$

1. **环境光 Ambient**
模拟场景多次反射形成的均匀背景光，不受光源角度影响：
$I_{ambient} = K_a \times C_{light} \times C_{object}$
$K_a$：环境光系数；$C_{light}$：光源颜色；$C_{object}$：物体固有色

2. **漫反射 Diffuse（朗伯模型）**
粗糙表面均匀散射光线，亮度由光线与法向量夹角余弦决定：
$I_{diffuse} = K_d \times \max(0, \mathbf{N} \cdot \mathbf{L}) \times C_{light} \times C_{object}$
$\mathbf{N}$：表面单位法向量；$\mathbf{L}$：指向光源的单位光线向量

3. **镜面高光 Specular**
光滑表面定向反射强光，高光范围由高光指数$n$控制：
$I_{specular} = K_s \times \max(0, \mathbf{R} \cdot \mathbf{V})^n \times C_{light}$
$\mathbf{R}$：光线反射单位向量；$\mathbf{V}$：指向相机的单位视线向量；$n$：高光光泽度 Shininess

### 2\.2 光线投射与深度测试原理

1. 屏幕每个像素发射一条射线，求解射线与球体、圆锥隐式几何体交点；

2. 深度测试 \(Z\-Buffer 逻辑\)：多条射线交点保留最小正$t$值（离相机最近），实现物体遮挡；

3. 获取最近交点的世界坐标与法向量，作为 Phong 着色计算输入。

## 五、代码整体逻辑与实验任务实现

### 任务 1：构建隐式三维场景（无外部模型）

1. 场景物体参数

    - 红色球体：圆心$(-1.2, -0.2, 0)$，半径 1\.2，固有色$(0.8,0.1,0.1)$，画布左侧；

    - 紫色圆锥：顶点$(1.2,1.2,0)$，底面$y=-1.4$，底面半径 1\.2，固有色$(0.6,0.2,0.8)$，画布右侧；

2. 全局固定参数

    - 相机射线起点：$(0,0,5)$

    - 点光源位置：$(2,3,4)$，光源纯白$(1,1,1)$

    - 画布背景色：深青色

### 任务 2：光线投射、几何体求交与深度测试

1. GPU 内核遍历全部像素，为每个像素生成一条投射射线；

2. 实现射线 \- 球体、射线 \- 圆锥数学求交函数，返回交点距离$t$；

3. 深度竞争逻辑：对比两个物体交点$t$，仅保留大于 0 的最小$t$，实现遮挡；

4. 根据最近交点$t$还原三维世界坐标，计算对应几何体表面单位法向量$\mathbf{N}$。

### 任务 3：Phong 着色器实现

1. 向量预处理：交点处计算指向光源向量$\mathbf{L}$、指向相机视线$\mathbf{V}$、反射向量$\mathbf{R}$，全部归一化为单位向量；

2. 分段计算三大光照分量，使用`ti.max(0, 点积结果)`截断负数，消除背面非法光照；

3. 叠加环境光、漫反射、高光得到原始 RGB 颜色，使用`ti.math.clamp`限制色彩区间$[0,1]$，防止颜色过曝发白；

4. 将最终合法颜色写入像素帧缓存。

### 任务 4：交互式 UI 控制面板

使用`ti.ui.Window`搭建渲染窗口，侧边绑定 4 个滑动条控件，实时同步参数至着色内核：

- 拖动滑块实时更新$K_a、K_d、K_s、Shininess$；

- 画面同步重渲染，直观观察每个参数对物体明暗、高光大小、反光强度的影响。

## 七、实验现象与结果分析

1. 参数调节现象

    - $K_a$增大：整体画面变亮，物体无光源照射区域也出现底色；$K_a=0$时背光面完全漆黑；

    - $K_d$增大：物体基础明暗对比更强，漫反射亮度提升；

    - $K_s$增大：镜面高光亮度变强；Shininess 数值越大，高光斑点越小、越锐利；数值越小高光越宽大模糊。

2. 遮挡与深度效果
球体与圆锥重叠区域自动实现遮挡，近处物体覆盖远处物体，验证深度测试逻辑正确。

3. 容错效果
代码对点积结果截断、色彩钳位，无黑色噪点、无画面发白溢出；所有参与光照计算向量统一归一化，不会出现全黑画面。

4. 拓展对比

    - Blinn\-Phong：高光计算速度更快，大入射角下高光区域比标准 Phong 更自然、范围更大；

    - 硬阴影：物体底部、相互遮挡区域生成纯黑色阴影，阴影区域仅保留环境光，立体层次感更强。

## 八、注意事项总结

1. 参与点积的$\mathbf{N}、\mathbf{L}、\mathbf{V}、\mathbf{R}$必须归一化为单位向量，否则会出现全黑画面；

2. 漫反射、高光点积结果必须用`ti.max(0, x)`截断负数，背面不产生光照，消除马赛克噪点；

3. 三通道色彩叠加后必须使用`clamp(0.0,1.0)`限制 RGB，防止数值超过 1 导致画面泛白过曝。
