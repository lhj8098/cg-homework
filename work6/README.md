## 一、实验目标

1. 理解可微光栅化原理，掌握离散网格三角形边界的平滑近似数学方法，解决硬光栅化梯度消失问题。

2. 掌握利用多视角二维剪影图像反向优化三维网格顶点坐标的实现流程。

3. 理解网格正则化的作用，知晓正则项如何避免网格拓扑崩坏、防止优化陷入局部最优。

## 二、实验原理

实验核心流程：以球体网格作为初始模型，通过梯度下降迭代形变，最终拟合出奶牛三维模型，过程解决两大核心问题。

### 1\. 软光栅化（解决梯度消失）

传统硬光栅化对像素做二值判断：像素在三角形内 / 外，边界为阶跃函数，梯度恒为 0，参数无法更新。
软光栅化引入像素到三角边缘距离，搭配 Sigmoid 生成平滑过渡权重：
$A(d) = \text{sigmoid}\left(\frac{d}{\sigma}\right)$
$\sigma$控制边缘模糊程度，边界内外均可产生有效梯度，指导顶点向目标轮廓移动。

### 2\. 网格正则化（防止局部最优与网格崩坏）

仅依靠剪影匹配损失会造成顶点交叉、网格拉伸、表面出现大量尖刺，陷入局部最优。引入三类正则损失约束网格形态：

1. 拉普拉斯平滑：约束相邻顶点，抑制表面尖锐凸起；

2. 边长惩罚：限制三角形边长，避免网格过度拉伸、压缩；

3. 法线一致性：约束相邻三角面片法线方向，维持曲面平滑。

总损失计算公式：
$L_{total} = L_{silhouette} + w_{lap}L_{lap} + w_{edge}L_{edge} + w_{normal}L_{normal}$
$L_{silhouette}$ 为剪影匹配损失，其余为加权正则约束损失。

## 三、实验任务与步骤

### 3\.1 环境配置

底层依赖 CUDA 算子，需要安装 torch、torchvision、pytorch3d。Windows 推荐 conda 安装，Mac 苹果芯片可直接编译适配版本。

### 3\.2 加载目标模型，生成参考剪影图

1. 导入奶牛目标网格模型；

2. 在三维空间布置多个均匀分布的相机；

3. 批量渲染多视角奶牛剪影图，作为优化监督真值。

### 3\.3 初始化模型与软渲染管线

1. 初始化高细分球体网格作为待优化初始模型；

2. 基于 PyTorch3D 搭建 SoftSilhouetteShader 软剪影光栅化渲染器。

### 3\.4 可微梯度下降优化循环

1. 将球体顶点偏移量 deform\_verts 设置为可求导参数 requires\_grad=True；

2. 前向渲染当前形变球体剪影，与目标剪影计算 MSE 匹配损失；

3. 在总损失中叠加拉普拉斯、边长、法线三类正则惩罚项；

4. 使用 Adam 或 SGD 优化器反向传播，更新顶点偏移参数。

### 3\.5 可视化输出

迭代过程中定期渲染、保存模型，直观查看球体逐步变形为奶牛的完整过程。

## 四、基础实现功能

1. 搭建软光栅化渲染管线，消除硬光栅化边界梯度消失问题；

2. 多视角剪影监督，实现球体网格逐步形变拟合奶牛轮廓；

3. 实现拉普拉斯、边长、法线三类网格正则约束，避免网格畸变；

4. 基于梯度下降完成端到端可微网格优化，支持训练过程可视化。

## 五、实验现象分析

1. 仅使用剪影损失、无正则：网格快速贴合轮廓，但顶点堆叠、面片拉伸扭曲，布满尖刺，陷入局部最优；

2. 增加正则约束：模型贴合外形的同时，网格保持均匀光滑，无拓扑错乱；

3. 软光栅化作用：训练全程存在有效梯度，模型持续平滑形变，不会出现训练停滞；

4. 迭代变化：随训练轮次增加，标准球体逐步凸起、凹陷，轮廓无限逼近奶牛形态。
